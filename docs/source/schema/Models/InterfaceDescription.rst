InterfaceDescription
---------------

Describes a low-level interface of an item.


.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
     - Remarks
     - Example

   * - description
     - String
     - A brief description.
     - optional, defaults to null
     - null
   * - format
     - String
     - The payload format.
     - optional, defaults to null
     - null
   * - url
     - String
     - A URL describing the endpoint.
     - optional, defaults to null
     - null
   * - protection
     - String
     - A description of the interface protection method.
     - optional, defaults to null
     - null
